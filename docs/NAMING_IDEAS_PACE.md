# 命名發想:以 Pace 為靈感

背景:目前專案名稱 Shuttle,核心產品是「用口語把行程變成時間軸」的行程規劃 App
(用自然語言記事 → AI 整理成事項/時間/地點 → 排上多軌時間軸 → 可邀同行者共同編輯、分享)。
以下是圍繞 **Pace**(步調、節奏、配速)發想、與產品概念的連結整理,尚未定案。

## 為什麼 Pace 跟這個專案有連結

- **旅程的節奏感** — Shuttle(接駁、往返)是「移動」的意象,Pace 補上「移動的快慢、步調」,
  合起來像「用你的步調移動」。
- **時間軸的本質就是 Pace** — 產品把行程排成有節奏感的多軌時間軸(主線行程 + 支線事項),
  正是「pace」的字面意義:事情發生的節奏。
- **語音輸入的講話步調** — 使用者說一句話、AI 立刻整理成行程,這種「講一句、馬上有結果」
  的體驗,也呼應 pace 的輕快、不拖泥帶水。

## 第一輪:直接複合詞根

| 名稱 | 組成 | 意象 |
|------|------|------|
| Pacer | Pace + -er | 配速員/領跑者;引申為「幫你抓好行程步調的人」,AI 是你的行程 pacer |
| PaceLine | Pace + Line | 借自自行車隊「破風隊形」術語,強調一群人保持節奏、一起前進,呼應時間軸(Line)與協作特性 |
| Tripace | Trip + Pace | 「行程的步調」,語意最精確對應產品定位,但是新造詞,需要時間讓人記住 |
| Journace | Journey + Pace | 同上邏輯,但較長 |

## 第二輪:更多方向

### A. 悠閒/渡假步調(呼應 landing page 現有「大海度假:海水藍綠 + 沙金」視覺主題)

| 名稱 | 組成 | 意象 |
|------|------|------|
| Lazepace | Lazy + Pace | 慵懶的步調,配色主題呼應度假海灘感 |
| Slopace | Slow + Pace | 放慢的步調,強調「旅行不用趕」 |
| Ripace | Retreat/Relax + Pace | 度假步調 |

### B. 一群人同步、共同前進(協作/邀朋友特性)

| 名稱 | 組成 | 意象 |
|------|------|------|
| Copace | Co(-together) + Pace | 一起走的步調,短、好記,呼應邀朋友共享行程的功能 |
| Wepace | We + Pace | 唸起來像 "we-pace",強調「我們的步調」 |
| Syncpace | Sync + Pace | 同步步調,較有技術感 |

### C. 口語輸入、一句話成行程的產品核心

| 名稱 | 組成 | 意象 |
|------|------|------|
| Sayace | Say + Pace | 說出來的步調(發音略牽強) |
| Voipace | Voice + Pace | 口語步調(發音拗口) |
| Talkpace | Talk + Pace | 講話步調,較白話直接 |

### D. 中英文雙關

| 名稱 | 組成 | 意象 |
|------|------|------|
| Pathce | Path + Pace | 路徑步調,發音近似 "pace",多一層「路徑/軌跡」意涵,呼應時間軸 |
| Pacego | Pace + Go | 步調出發,唸起來輕快,適合口語化品牌 |
| 步程(中文)+ Pace(英文) | 步(呼應 Pace)+ 程(呼應行程/旅程) | 中文用既有詞彙組合而非硬造字,自然好唸,可搭英文 Pace 系列品牌 |

### E. 更簡短有力,去掉字根痕跡

| 名稱 | 組成 | 意象 |
|------|------|------|
| Pacen | Pace + n | 尾音加重,唸起來乾脆,像品牌名而非合成詞 |
| Pacery | Pace + -ery | 仿 bakery 造詞,暗示「一個做步調規劃的地方/服務」 |
| Onpace | 借用英文成語 "on pace" | 意指「跟上進度/節奏」,直接借用有意義的既有片語當品牌名,比造字更自然好記,且本身帶正面的順遂、愉快意涵 |
